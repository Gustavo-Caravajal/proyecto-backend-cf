API BOOKS

Version 1

solo permite agregar autores, estos se agregan con el siguiente json de ejemplo:

{"full_name": "nombre", "birth_year": 1970, "country_origin": "pais", "years_experience": 15}









autores

Version 2

ademas de autores, permite agregar editoriales, estas ultimas se agregan con el siguiente json de ejemplo:

{"name": "editorial", "founded_year": 1950}










autores editoriales

Version 3

ademas de autores y editoriales, permite agregar libros, estos ultimos se agregan con el siguiente json de ejemplo:

{"title": "libro", "publication_year": 2000, "n_pages": 500,

"languaje": "idioma",

"author_id": 1,

"publisher_id": 1}





autores editoriales libros

Version 4

ademas de autores, editoriales y libros, permite agregar generos, estos ultimos se agregan con el siguiente json de ejemplo:

{"name": "genero"}

y el libro en esta version se agrega con el siguiente json:

{"title": "libro", "publication_year": 2000, "n_pages": 500,

"languaje": "idioma",

"author_id": 1,

"publisher_id": 1,

"genre_id": 1}

autores editoriales libros generos
